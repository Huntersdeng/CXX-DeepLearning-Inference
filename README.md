# CXX-DeepLearning-Inference

## Introduction
A unified and extensible pipeline for deep learning model inference with C++.
### Support framework
- [x] ONNXRuntime
- [x] TensorRT
- [ ] RKNN
### Support model
- object-detection
  - [x] yolov8 (det, seg)
- ocr
  - [x] crnn
  - [x] abinet
  - [ ] dbnet
